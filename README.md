## О приложении

**FinTrack** помогает держать в одном месте все ваши кошельки — карты, вклады, наличные, накопительные счета — 
и видеть, как меняется общий баланс со временем. Данные хранятся локально в `localStorage`. Бэкап и перенос между 
устройствами — через экспорт/импорт JSON.

> Сервис заточен под использование **на телефоне**. Его удобно добавить на домашний экран (особенно на iOS) и пользоваться как обычным приложением — без адресной строки и в полный экран. См. раздел [«Установка на телефон»](#-установка-на-телефон-как-приложение).

## Возможности

- **Источники средств** — добавляйте, редактируйте и удаляйте «кошельки» с названием, суммой и собственным цветом.
- **Кольцевая диаграмма** — наглядное распределение баланса по источникам с общей суммой в центре.
- **Динамика по месяцам** — баланс на конец каждого месяца сохраняется автоматически и отображается на графике.
- **Тёмная и светлая темы** — переключаются в настройках, выбор запоминается.
- **Два языка** — русский и английский (i18n с локализацией дат и чисел).
- **Бэкап данных** — экспорт и импорт в JSON с валидацией файла.
- **Локальное хранение** — всё работает офлайн, данные остаются в браузере.
- **Плавные анимации** — на базе Framer Motion.

## Технологии

- **React 19** + **TypeScript**
- **Vite 7** — сборка и dev‑сервер
- **Motion** — анимации (`LazyMotion` / `strict`)
- **lucide-react** — иконки
- **CSS Modules** + дизайн‑токены, единая палитра в `src/theme/colors.ts`
- **localStorage** — персистентность

## Запуск

Нужен **Node.js 18+** и любой пакетный менеджер (примеры на `npm`).

```bash
# установка зависимостей
npm install

# запуск dev-сервера (по умолчанию http://localhost:5173)
npm run dev

# сборка production-версии в dist/
npm run build

# локальный предпросмотр собранной версии
npm run preview

# линтинг
npm run lint
```

## Установка на телефон как приложение

FinTrack — это статический сайт, поэтому его можно «установить» прямо из браузера и запускать в 
полноэкранном режиме, как нативное приложение.

**iOS (Safari):**
1. Откройте сайт в Safari.
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой вверх).
3. Выберите **«На экран Домой»** → **«Добавить»**.
4. Запускайте FinTrack с домашнего экрана — он откроется без адресной строки.

**Android (Chrome):**
1. Откройте сайт в Chrome.
2. Меню (⋮) → **«Добавить на главный экран»**.

После сборки (`npm run build`) папку `dist/` можно развернуть на любом статическом хостинге (GitHub Pages, Netlify, Vercel, Cloudflare Pages) и открыть с телефона.

## Данные и бэкап

- Все данные хранятся локально в `localStorage` текущего браузера.
- В разделе **Настройки → Данные** можно выгрузить резервную копию (`fin-track-backup-YYYY-MM-DD.json`) и восстановить её на другом устройстве или в другом браузере.
- Файл бэкапа валидируется при импорте — посторонние или повреждённые файлы будут отклонены.
